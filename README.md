Ülesanne

Tee uus projekt (nimes peab olema sinu eesnimi). Loo GUI lahendus järgmise asja jaoks.
Aknal on silt (Label), sisestuaskast (Entry) ja tulemuskast (Text).
Lisaks kaks nuppu: Näita, Puhasta.

Sisestuskastis loetakse info ning näidatakse vastavalt väärtusele järgnevat infot:
- Paaris või paaritu
- Postitiivne või Negatiivne
- Täisarv või Murdarv
- Jagub kolmega või ei jagu kolmega
- Ümmargune number või mitte (ümmargune on 5 või 0 lõppevad numbrid. 5, 10, 15 jne)

- Tühi või mitte
- Algab suure tähega või mitte
- Sisaldab numbrit või mitte
- Üks või mitu sõna
- Palindrom või mitte (mõlemat pidi lugedes on sama sõna. Näiteks LEVEL)

Seega sõltumata sisestusest peab ta midagi näitama iga asja kohta kõiki neid asju.
Kui vajutati nupule Näita, siis tehakse ka sisestuskast tühjaks (automaatselt).
Vastuses peab näitama lisaks eelnevale ka mida sisestati sisestuskasti.
Pythoni lisa mooduleid ei paigalda.

Proovige luua GUI OOP (Objekt-Orienteeritud-Programmeeimine) põhimõttel ehk looge klass GUI jaoks.
Looge klass ka tulemuste saamiseks.
 Mõtle üks samm ette, äkki on vaja kuhugi midagi lisaks lisada või midagi uut sarnase loogikaga lisada.
